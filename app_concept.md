# 旅費規程自動化ツール - アプリコンセプト

## 概要

「旅費規程運用を自動化する個人向けツール」

**目的**: 日当・交通費の管理と税務的に強い記録を"勝手に作る"  
**対象**: 旅費規程を使っている一人社長・フリーランス

---

## コア価値

手入力の手間を減らすことではなく、**"税務的に通る状態を自動で作る"**こと。

- 記録漏れ防止
- 日当の自動計算
- 証拠ログの蓄積
- 会計ソフト連携

この4つを一気通貫でやる。

---

## 機能（MVP）

### 入力項目（極限までシンプルに）

| 項目 | 内容 |
|------|------|
| 日付 | |
| 場所 | |
| 内容 | 交流会 / 主催 / 商談 など選択式 |
| 時間 | ざっくりでOK |

### 自動処理（入力後に裏側でやること）

- **日当の自動判定**: 時間・内容に応じて 3,000円 or 5,000円 を自動計算
- **交通費**: 簡易入力 or 定型登録
- **月次集計**: 月ごとの合計を自動集計
- **ログ生成**: 「いつ・どこで・何をしたか」を税務説明できる形で保存

---

## 記録強化（差別化ポイント）

ここがプロダクトの芯。「ただの入力ツール」ではなく**証拠生成ツール**にする。

- 参加者メモ（任意）
- 主催 or 参加フラグ
- 簡単なメモ欄
- 位置情報（可能なら）

---

## 出力

**freee / マネーフォワード向け CSV 出力**（ワンクリックでそのままインポート可能な状態）

| カラム | 内容 |
|--------|------|
| 日付 | |
| 金額 | |
| 勘定科目 | 旅費交通費 |
| 摘要 | 交流会・主催 など |
| 税区分 | |

---

## 運用フロー

```
その場で1分入力
↓
月末にCSV出力
↓
freeeにインポート
↓
完了
```

---

## 拡張余地

- 定期イベントテンプレ
- 参加者管理
- 売上（案件化）との紐付け
- SNSログ連携
- 税理士共有機能

---

## ポジション

会計ツールではなく、**"お金を非課税で抜くための実務ツール"**

---

## GAIさんとの相性

- 主催頻度が高い
- 外出が多い
- すでに旅費規程を使う前提がある

この3つが揃っているため、まず自分用として価値が成立する。

**成長ロードマップ**: 自分で使う → 改善する → 他人にも刺さる形にする
